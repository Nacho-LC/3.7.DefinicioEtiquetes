Se usa para indicar el título de una obra, como un libro, una película, un artículo, etc. Generalmente, los navegadores lo muestran en cursiva.Ejemplo:

Uno de mis libros favoritos es El principito.

Se usa para incluir una cita breve dentro del texto. Los navegadores suelen envolver el contenido entre comillas automáticamente.Ejemplo:

Einstein dijo: La imaginación es más importante que el conocimiento.

Se usa para definir un término que será explicado en el mismo contexto. Por convención, los navegadores suelen mostrar el contenido en cursiva.Ejemplo:

El término HTML significa HyperText Markup Language.

Se usa para definir una abreviatura o acrónimo. Puede incluir el atributo title, que muestra el significado completo al pasar el cursor sobre el texto.Ejemplo:

El protocolo HTTP es fundamental para la web.
